## 1. owner question

`Coding Pack` 作为团队能力面，应该如何被组织和使用。

## 2. 设计判断

- 它不是单一 coding tool
- 它是 team capability family
- 它要和 review / evidence / growth 主线协作

## 3. 核心设计对象

- roles
- workflows
- artifacts
- review boundaries
- continuity handoff points
