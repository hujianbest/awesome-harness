# 上游工件摘要: 异步 Standup 工具

## Concept Brief
- Recommended Wedge: 结构化文字替代语音 standup，自动从项目管理工具拉取进度
- 目标用户: 5-15 人远程团队

## Key Findings
- 团队最痛苦的是 standup 开会占时间（Observed）
- 自动拉取项目管理工具数据技术上可行（Observed, GitHub/Jira API 都支持）
- 用户是否真的愿意每天写 3 分钟文字仍不确定（Untested）

## Differentiation
- 与 Standuply / Geekbot 的区别: AI 总结 + 自动拉取而非只发问卷
